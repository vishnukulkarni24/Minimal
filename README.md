**NextStore — Minimalist E-Commerce Web App**

NextStore is a full-stack e-commerce website built using **Next.js**, **React.js**, and **Express.js** with a clean, minimalist UI. It includes all essential shopping features like products, categories, cart, filters, checkout, and invoices.
<img width="1815" height="902" alt="image" src="https://github.com/user-attachments/assets/2e596a4f-6162-43b6-8bf8-2b66b437c1cf" />

**Features:**

* Product listing, details, and cart management
* Filters by category, price, and keyword
* Dummy checkout (Card, PayPal, COD)
* Invoice generation after checkout
* Responsive minimalist UI with Tailwind CSS
* Context API for global state
* Mock database and data seeding

**Tech Stack:**
Next.js, React.js, Node.js, Express.js, TypeScript, Tailwind CSS

**Setup:**

1. Clone repo → `git clone https://github.com/vishnukulkarni24/Minimal.git`
2. Install deps → `npm install`
3. Run → `npm run dev`
4. Visit → `http://localhost:5000`
