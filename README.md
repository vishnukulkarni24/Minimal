NextStore — Minimalist E-Commerce Web App

OVERVIEW:
NextStore is a full-stack e-commerce website built with Next.js, React.js, and Express.js featuring a clean, minimalist UI.
It includes all the essential shopping functionalities such as product listings, categories, cart management, filters, checkout, and invoice generation.
The project is designed for scalability, clarity, and modern web performance.

MAIN FEATURES:

Core Shopping Features:

Product Listing Page: Displays all products with category and price filters.
Product Details Page: Shows full details of a selected product with an “Add to Cart” option.
Shopping Cart: Lets users update quantity, remove items, and view totals.
Checkout Page: Simulates payment with dummy options like Card, PayPal, and Cash on Delivery.
Invoice Page: Displays order summary and total amount after checkout.
Filters and Search: Filter products by category, price, or keyword.

Technical Highlights:

Built using Next.js and React for fast rendering and SEO optimization.
Express.js handles backend routes and dummy data.
TypeScript ensures clean, type-safe code.
Tailwind CSS provides a responsive, minimalist design.
Context API or Redux manages cart state globally.
Includes a mock database and data seeder for dummy content.
Vite integrated for faster builds and development.

INSTALLATION & SETUP:

Prerequisites:

Node.js (v18 or higher)

npm or yarn

Steps:

Clone the repository
git clone https://github.com/vishnukulkarni24/Minimal.git

Navigate to the project folder
cd Minimal

Install dependencies
npm install

Run the development server
npm run dev

Open your browser and visit
http://localhost:5000

TECH STACK:

Frontend: React.js, Next.js
Backend: Node.js, Express.js
Language: TypeScript
Styling: Tailwind CSS
State Management: React Context API
Data: Dummy JSON (mock seeding)
